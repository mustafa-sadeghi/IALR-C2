# نسخه وب پروپوزال IALR-C2

این پوشه برای انتشار مستقیم روی GitHub Pages آماده است.

## فایل‌ها

- `index.html`: ساختار کامل صفحه و تمام متن پروپوزال
- `style.css`: طراحی واکنش‌گرا، حالت روشن/تیره و استایل چاپ
- `script.js`: فهرست موبایل، جست‌وجوی فهرست، نوار پیشرفت و تغییر پوسته

## انتشار در GitHub Pages

1. یک مخزن جدید در GitHub بسازید.
2. هر سه فایل را در شاخه اصلی مخزن قرار دهید.
3. از مسیر **Settings → Pages**، گزینه **Deploy from a branch** را انتخاب کنید.
4. شاخه `main` و پوشه `/root` را انتخاب و ذخیره کنید.
5. پس از انتشار، آدرس سایت در همان صفحه نمایش داده می‌شود.
