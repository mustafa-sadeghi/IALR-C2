# نسخه حرفه‌ای وب پروپوزال IALR-C2

این پروژه برای انتشار مستقیم روی **GitHub Pages** آماده شده و تمام متن پروپوزال را در یک رابط کاربری فارسی، راست‌چین و واکنش‌گرا نمایش می‌دهد.

## امکانات نسخه جدید

- طراحی مدرن برای دسکتاپ، تبلت و موبایل
- فهرست مطالب ثابت با نمایش فصل و زیرعنوان فعال
- جست‌وجوی سریع در کل سند با میانبر `Ctrl + K` یا `⌘ + K`
- حالت روشن و تاریک
- تنظیم اندازه متن و فاصله خطوط
- حالت تمرکز برای مطالعه بدون عناصر جانبی
- پیمایش فصل قبلی و بعدی
- نمایش درصد پیشرفت مطالعه و زمان تقریبی مطالعه
- کپی پیوند مستقیم هر فصل و زیرعنوان
- نسخه بهینه برای چاپ و ذخیره به‌صورت PDF
- بدون وابستگی به کتابخانه یا سرویس خارجی

## ساختار فایل‌ها

```text
index.html
style.css
script.js
README.md
assets/
├── favicon.svg
└── fonts/
    └── README.txt
```

## فعال‌سازی BTitr و BNazanin

فایل‌های فونت به‌دلیل حقوق نشر داخل پروژه قرار نگرفته‌اند. فایل‌های قانونی خودتان را در مسیر زیر قرار دهید:

```text
assets/fonts/BTitr.woff2
assets/fonts/BNazanin.woff2
```

قالب TTF نیز پشتیبانی می‌شود:

```text
assets/fonts/BTitr.ttf
assets/fonts/BNazanin.ttf
```

نام فایل‌ها باید دقیقاً مطابق بالا باشد. تیترها با `BTitr` و متن اصلی با `BNazanin` نمایش داده می‌شوند. در صورت نبودن فونت‌ها، فونت جایگزین سیستم فعال خواهد شد.

## جایگزینی نسخه فعلی در GitHub

1. فایل‌های نسخه قبلی را در ریشه مخزن با فایل‌های این پروژه جایگزین کنید.
2. پوشه `assets` را نیز به‌طور کامل آپلود کنید.
3. تغییرات را با پیامی مانند زیر Commit کنید:

```text
Upgrade proposal UI and reading experience
```

4. وارد تب **Actions** شوید و منتظر علامت سبز انتشار بمانید.
5. GitHub Pages معمولاً چند دقیقه بعد نسخه جدید را نمایش می‌دهد.
6. اگر ظاهر قبلی باقی ماند، صفحه را با `Ctrl + F5` بازخوانی سخت کنید.

## انتشار اولیه در GitHub Pages

از مسیر زیر:

```text
Settings → Pages → Deploy from a branch
```

شاخه `main` و پوشه `/(root)` را انتخاب کنید.
